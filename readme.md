# ESOLang
EsoLang is an ESOteric Language written in Python.
# Markers
`(` start<br>
`)` end/print<br>
`>` go right<br>
`<` go left<br>
`^` go up<br>
`v` go down<br>
`/` check next cell<br>
`\` check previous cell<br>
# Usage
`python3 esol.py <file.esol> (direction)`<br>
directions: `up`,`right`,`down`,`left`
